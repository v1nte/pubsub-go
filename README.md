# PubSub Patern in Go

Implementation of Publisher and Suscribers patter in Go

## Features

- [x] Subscribe
- [x] Unsubscribe
- [x] Unsubscribe all
- [x] Send msg via Broker
- [x] Handle multiples clients
- [x] Include name in messages
- [x] Use channels
- [x] Dockerize
- [ ] Logs
- [x] DB connection
- [ ] Auth
